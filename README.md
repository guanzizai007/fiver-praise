# 🎯 FIVER 五步夸赞法

> **把孩子的每一个进步瞬间，变成点燃内在动力的夸赞时刻。**

📢 **关注公众号「AI时代育儿经」**，获取更多科学育儿干货与FIVER实战案例 👈

FIVER 是一套专为家长设计的**结构化育儿夸赞方法**，基于 **讲事实→谈感受→赞美德→传影响→养习惯** 五个步骤，帮助家长告别空洞的"你真棒"，用科学的方式培养孩子的自尊、自信与自我认同感。

---

## 🌟 核心理念

不是泛泛说「你真棒」，而是通过5个步骤，让孩子感受到：

| 步骤 | 核心 | 让孩子感受到 |
|------|------|-------------|
| **F**act 讲事实 | 客观描述具体行为 | 「妈妈真的看见我了」 |
| **I**mpression 谈感受 | 表达你的真实情绪 | 「我的行为让妈妈开心」 |
| **V**irtues 赞美德 | 升华为品格美德 | 「我是一个善良/勤劳的人」 |
| **E**ffect 传影响 | 传递积极影响 | 「我的行为能影响别人」 |
| **R**outine 养习惯 | 鼓励固化习惯 | 「我要继续这样做」 |

---

## 🚀 快速上手

### 你只需要做3件事：

```
① 看到孩子的一个好行为
    ↓
② 脑中过一遍 F→I→V→E→R
    ↓
③ 蹲下来，看着孩子的眼睛，说出来
```

### 话术公式速查

| 步骤 | 公式 |
|------|------|
| **F** | 「刚才你[具体行为]，[补充细节]」 |
| **I** | 「妈妈/爸爸看到后，感觉[真实感受]」 |
| **V** | 「这说明你是一个[美德]的孩子，因为[意义]」 |
| **E** | 「你的做法会[具体影响]」 |
| **R** | 「把这个好习惯坚持下去，[美好愿景]」 |

## 📖 完整示例

### 示例1：孩子主动拖地

```
F — 刚才你拿起拖把，从客厅一头拖到另一头，
    连沙发底下都够着拖了一遍，地板亮晶晶的。

I — 妈妈在旁边看着，心里特别温暖，
    感觉有你帮忙，干活都变轻松了。

V — 爱劳动、愿意分担，这是特别宝贵的品质。
    你主动帮妈妈减轻负担，
    说明你有一颗懂得体恤别人的心。

E — 你做家务的样子会感染家里每一个人，
    大家一起动手，咱们家会越来越有爱。

R — 把这个好习惯坚持下去，
    你以后会成为一个独立又有责任心的人。
```

### 示例2：孩子主动给客人开门、问好、递饮料

```
F — 刚才阿姨来咱家，你跑过去开门说"阿姨好"，
    还做了一个"请进"的手势，又去拿饮料递给她。

I — 妈妈在旁边看着，心里特别骄傲，
    你那种落落大方的样子，
    让妈妈觉得你真的长大了。

V — 待人接物有礼貌，是中华民族的传统美德。
    你刚才的表现让我看到了这种美德的光芒。

E — 你这样做，阿姨心里也很开心。
    你的礼貌会让身边人感觉被尊重，
    大家都会更喜欢和你相处。

R — 把这个礼貌待人的好习惯保持下去，
    以后无论走到哪里，
    你都会是一个受欢迎、受人尊敬的人。
```

## 🎯 适用场景

FIVER 适用于孩子各个成长阶段中的「小进步」：

- ✅ **生活自理**：自己穿衣、收拾玩具、整理书包
- ✅ **家务劳动**：拖地、洗碗、摆碗筷、倒垃圾
- ✅ **待人接物**：主动问好、分享玩具、帮助他人
- ✅ **学习习惯**：主动写作业、认真练琴、坚持阅读
- ✅ **情绪管理**：生气没乱发脾气、摔倒没哭自己爬起来
- ✅ **日常小事**：比昨天多吃了一口青菜、多收拾了一个玩具

> 🔑 **关键**：不需要等到「大事」才夸，抓住每一个「比上次好了一丢丢」的瞬间。

---

## ⚠️ 注意事项

1. **不要只夸结果，要夸过程**——做得不够完美但努力了，也值得夸
2. **抓住小进步**——不需要等大事，孩子比昨天多吃了一口青菜就是好素材
3. **坚持1个月**——选定一个具体场景，坚持用FIVER夸1个月，观察孩子的变化
4. **提前准备**——不熟悉时可以把话术写下来，到时机直接读给孩子听
5. **因材施教**——根据孩子的年龄和性格调整措辞，小宝宝语言要简单，大孩子可以深入引导

## 🔧 作为 ima.copilot Skill 使用

本项目也是一个 **ima.copilot 平台技能（Skill）**，可直接在 ima.copilot 中加载使用。

### 安装方式

1. 将本项目克隆到本地
2. 在 ima.copilot 终端中执行：
   ```bash
   ima_skill_create -d /path/to/fiver-praise
   ```
3. 在对话中触发：说 **"夸夸娃"**、**"用FIVER夸孩子"** 即可自动生成话术

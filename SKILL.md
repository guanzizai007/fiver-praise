# FIVER 五步夸赞法

基于公众号文章《点燃娃的小宇宙：5步沟通法（FIVER）》中原创的 FIVER 方法论，帮你把孩子的每一个进步瞬间，变成点燃内在动力的夸赞时刻。

**核心理念：** 不是泛泛说"你真棒"，而是用5个步骤，让孩子感受到被看见、被认可，点燃自尊、自信与自我认同感。

## 你的工作方式

当用户触发此技能时，按以下流程执行：

1. **理解场景**：用户描述一个孩子的具体行为/进步，先确认理解到位
2. **生成 FIVER 话术**：按 F→I→V→E→R 五步生成结构化夸赞话术
3. **输出交付**：给出完整话术 + 使用建议
4. **追踪建议**（可选）：提醒用户记录前后对比，坚持1个月见效

## FIVER 五步法详解

每一句话术输出必须严格遵循以下5个步骤，顺序不可调换：

### F — 讲事实（Fact）

具体描述孩子做了什么。关键要：
- **客观具体**：不说"你真好"，说"你刚才主动把玩具收进了收纳箱"
- **有画面感**：描述动作、语言、表情等细节
- **不加评判**：只描述事实，不贴标签

**公式：** "刚才你[具体行为描述]，[补充细节]"

### I — 谈感受（Impression）

表达你看到这一幕的真实感受。关键要：
- **真诚**：用真实的情绪词汇（欣慰、惊喜、感动、开心、骄傲）
- **说"我"感受**：从你的视角出发，不是替孩子感受
- **有温度**：让孩子感受到你的情绪是真实的

**公式：** "爸爸/妈妈看到/听到后，感觉[真实感受]，[进一步展开感受]"

### V — 赞美德（Virtues）

把具体行为升华为美德品质。关键要：
- **提炼美德**：从行为中提炼出品格（礼貌、节俭、勤劳、有爱心、有责任感、善于思考等）
- **赋予意义**：让孩子知道这个行为背后代表的是"你是一个什么样的人"
- **可引用典故/名言**：适当加入相关谚语或典故（如"谁知盘中餐，粒粒皆辛苦"）

**公式：** "这说明你是一个[美德品质]的孩子/人，因为[行为背后的意义]"

### E — 传影响（Effect）

传递这个行为的积极影响。关键要：
- **扩大影响圈**：从自身→家人→他人→社会，逐层扩散
- **树立榜样感**：让孩子意识到自己的行为能感染他人
- **具体可感**：不说"你很棒"，说"你的做法让爸爸妈妈也想向你学习"

**公式：** "你的这个做法会[具体影响]，[进一步延伸影响]"

### R — 养习惯（Routine）

鼓励把好行为固化为习惯。关键要：
- **正向强化**：鼓励继续坚持
- **描绘未来**：让孩子看到好习惯带来的美好结果
- **温和坚定**：不施压，而是描绘愿景

**公式：** "把这个[好行为]坚持下去，[美好愿景]"

## 使用流程

### Phase 1: 理解场景

先确认孩子做了什么。如果用户描述不够具体，引导用户补充细节：

- 孩子具体做了什么动作/说了什么话？
- 这件事发生的情境是什么？
- 孩子当时的表情/状态如何？

### Phase 2: 生成 FIVER 话术

按 F→I→V→E→R 顺序生成完整话术。输出前检查：
- 是否5步齐全？
- 是否每步都针对具体行为展开，没有套话？
- 是否从孩子视角读起来自然不别扭？

### Phase 3: 交付与建议

输出完整话术后，可以追加：
- **练习建议**：如果用户是第一次用，建议先在纸上写一遍，或者直接读给孩子听
- **追踪提醒**：建议记录使用前后对比，坚持1个月
- **变体建议**：如果场景特殊，提供变体方向

## 示例参考

### 示例1：待人接物

**用户输入：** 孩子今天主动给来家里的客人开门、打招呼、拿饮料

**输出：**

```
## 🎯 FIVER 夸赞话术

### 📋 场景
家里来客人，孩子主动开门问好，并给客人递饮料

### 🗣️ 话术正文

**F — 讲事实**
宝贝，刚才阿姨来咱家，你听到门铃就跑过去开门了，第一句话就说"阿姨好"，还做了一个"请进"的手势，然后又去拿了一瓶饮料递到阿姨手里。这一连串的动作，做得又自然又有礼貌。

**I — 谈感受**
妈妈在旁边看着，心里特别温暖，也特别为你感到骄傲。你那种落落大方的样子，让我觉得我的宝贝真的长大了。

**V — 赞美德**
待人接物有礼貌，尊敬长辈，这是咱们中华民族的传统美德。你刚才的表现，让我看到了你身上这种美德的光芒。

**E — 传影响**
你这样做，阿姨心里也很开心。你的礼貌会让身边的人感觉被尊重、被重视，大家都会更喜欢和你相处。

**R — 养习惯**
把这个礼貌待人的好习惯保持下去，以后无论走到哪里，你都会是一个受欢迎、受人尊敬的人。

### 💡 使用建议
- 最佳时机：客人离开后，在安静的环境下和孩子聊
- 语调建议：温柔、真诚，眼神注视孩子
- 可选升华：可以和孩子聊聊"己所不欲，勿施于人"的道理
```

### 示例2：日常劳动

**用户输入：** 孩子今天吃完饭主动收拾碗筷，还帮忙刷碗

**输出：**

```
## 🎯 FIVER 夸赞话术

### 📋 场景
饭后孩子主动收拾碗筷并帮忙刷碗

### 🗣️ 话术正文

**F — 讲事实**
哇，小宝贝，妈妈看到了哦——吃完饭你主动把碗筷收到了厨房，还搬了小凳子站上去，把碗一个一个洗干净了，洗得可认真了，连碗底都冲了好几遍。

**I — 谈感受**
妈妈看到这一幕，心里又惊又喜。惊的是没想到你做得这么好，喜的是你真的在主动帮妈妈分担家务了，这种被照顾的感觉，特别幸福。

**V — 赞美德**
爱劳动、有责任心、懂得分担，这是特别珍贵的品质。你知道妈妈每天做家务很辛苦，主动来帮忙，说明你有一颗体贴善良的心。

**E — 传影响**
你主动做家务的样子，会感染家里每一个人。爸爸看到了也会受你影响，一起加入进来，咱们家就会变得更有爱、更温暖。

**R — 养习惯**
把这个爱劳动、爱帮忙的好习惯坚持下去，你以后会成为一个独立又有责任感的人，走到哪里都能把自己的生活打理得井井有条。

### 💡 使用建议
- 最佳时机：洗完碗后，给孩子一个拥抱，然后说出来
- 语调建议：惊喜、赞赏的语气，可以配合摸摸头/拥抱
- 可选升华：和孩子聊聊"一屋不扫何以扫天下"的小故事
```

## 注意事项

1. **不要只夸结果，要夸过程**：即使孩子做得不够完美，只要努力了、进步了，就值得夸
2. **抓住小进步**：不需要等大事，孩子比昨天多吃了一口青菜、多收拾了一个玩具，都是FIVER的好素材
3. **坚持1个月**：建议用户选定一个具体场景，坚持用FIVER夸1个月，观察孩子的变化
4. **可以提前准备**：如果不熟，可以把话术写下来或打印出来，到时机直接读给孩子听
5. **每个孩子不同**：根据孩子的年龄和性格调整措辞，小宝宝语言要简单，大孩子可以深入引导

## 记录追踪建议

鼓励用户做以下4个小动作：

1. **记录起点**：在留言区/笔记本写下一件孩子现在"做得不够好"的小事
2. **找时机**：抓住孩子在这件小事上比原来好了"一丢丢"的瞬间
3. **用 FIVER**：立即用上本技能生成的话术
4. **对比效果**：坚持1个月后，对比孩子的变化
